Inspection API + Indexing API · Google Search Console

De tool werkt in twee fases: inspecteren (wat weet Google van deze URL?) en indienen (recrawl-verzoek sturen). Daarna zie je drie tabs met de resultaten.

1
Twee APIs inschakelen in Google Cloud

Ga naar console.cloud.google.com → API's & Services → Bibliotheek en schakel beiden in:

1. Google Search Console API (inspecteren)
2. Web Search Indexing API (indienen)
2
Service Account + JSON key

Maak een service account aan via Inloggegevens → Service-account aanmaken. Download daarna een JSON sleutelbestand via Sleutels → Sleutel toevoegen → JSON.

🔒Bewaar dit bestand veilig — deel het nooit.
3
Service Account als Eigenaar in Search Console

GSC → property → Instellingen → Gebruikers → Gebruiker toevoegen. Rol: Eigenaar (verplicht voor beide APIs).

📧 naam@project.iam.gserviceaccount.com
🔑 Rol: Eigenaar
4
Indiendatum gebruiken voor "Nog wachtend"

Vul je de originele indiendatum in (bv. de datum van een vorige run), dan markeert de tool URL's die 14+ dagen geleden zijn ingediend en nog steeds niet geïndexeerd zijn als "Nog wachtend". Zo zie je direct welke URL's echt aandacht nodig hebben.

Kan ik een ban of penalty krijgen?
🔍 Inspection API — volledig veilig
Officieel door Google gebouwd voor bulk-inzicht in indexeringsstatus voor je eigen property. Lezen geeft nooit een penalty.
📤 Indexing API — veilig, met nuance
Google geeft HTTP 200 terug ook voor reguliere pagina's. De API is officieel voor JobPosting/BroadcastEvent, maar er staat geen straf op gebruik voor andere pagina's — Google negeert het signaal simpelweg of prioriteert het lager. Een ban of penalty krijg je van manipulatie (spam, cloaking, gekochte links), niet van API-gebruik op je eigen site.
Waarom worden niet alle URL's geïndexeerd?
URL's met status "Onbekend bij Google" zijn waarschijnlijk niet bereikbaar via interne links of een sitemap. De Indexing API helpt daar weinig. Zorg eerst dat die pagina's vindbaar zijn, dan werkt indienen beter.
Quota's
Inspection: 2.000/dag per property. Indexing: 200/dag standaard (uitbreidbaar via Google Cloud Console).
